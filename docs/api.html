GET /search?q={query}
Search postal codes by location name. 100% compatible with kodepos.vercel.app

Example:

GET /detect?latitude={lat}&longitude={lng}
Detect location by coordinates. Returns nearest postal code with distance.

Example:

curl "https://your-api.workers.dev/detect?latitude=-6.2088&longitude=106.8456"
GET /api/v1/search?{filters}
Advanced search with multiple filters. Search by province, city, district, etc.

Example:

curl "https://your-api.workers.dev/api/v1/search?provinsi=DKI Jakarta&kota=Jakarta Pusat"
GET /api/v1/nearby?lat={lat}&lng={lng}&radius={km}
Find postal codes within specified radius of coordinates.

Example:

curl "https://your-api.workers.dev/api/v1/nearby?latitude=-6.2088&longitude=106.8456&radius=5"
